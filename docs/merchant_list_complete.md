## Adding Custom Merchants

Users can add their own merchants through the application:

```python
from components.merchant_db import add_custom_merchant

add_custom_merchant(
    session=session,
    model_class=Merchant,
    name="YOUR MERCHANT NAME",
    category="Category",
    txn_type="Expense",
    aliases=["ALIAS1", "ALIAS2"],
    is_personal=False,
    industry="Industry",
    confidence_boost=25
)
```

## Notes

1. All merchant names are stored in UPPERCASE
2. Aliases include common variations and misspellings
3. Confidence boost ranges from 0-30
4. Personal flag indicates typically personal (non-business) transactions
5. Default categories can be overridden by users
6. The database is expandable - users can add custom merchants
